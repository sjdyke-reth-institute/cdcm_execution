from cdcm import *

State#

State is a Variable whose value changes with time. Change in discrete time step is considered here. Two versions of value are stored for a State. The value of state at current time step is State.value and the value at next time step is State.next_value.

Creating a state#

s = State(
    value=0.5,
    units="m",
    track=True,
    description="A standard state."
)
print(s)

unamed_node:
  description: A standard state.
  owner: ''
  parents: ()
  children: ()
  value: 0.5
  units: m
  track: true

print(f"The present value of the state is: {s._value}")

The present value of the state is: 0.5

print(f"The next value of the state is: {s._next_value}")

The next value of the state is: 0.5

s._next_value = 1.5
print("Changing the next value.")
print(f"The next value of the state is: {s._next_value}")

Changing the next value.
The next value of the state is: 1.5

print("Swaping values.")
s.transition()
print(s)

Swaping values.
unamed_node:
  description: A standard state.
  owner: ''
  parents: ()
  children: ()
  value: 1.5
  units: m
  track: true

transition is a function which swaps the values of State.value and State.next_value. More on transition function will be discussed later.

Another way of creating a State#

Syntax: make_node(“S:name:value:unit”)

x = make_node("S:x:1:m")
print(x)

x:
  description: ''
  owner: ''
  parents: ()
  children: ()
  value: 1
  units: m
  track: true
